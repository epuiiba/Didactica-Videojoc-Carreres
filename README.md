# 🎮 Situació d'Aprenentatge: Desenvolupament d'un Videojoc de Carreres 3D en Unity

Aquesta situació d'aprenentatge és una proposta educativa per al mòdul professional de **“Programació i motors de videojocs”** del curs d'espelització **"Desenvolupament de videojocs i realitat virtual"**. Aquest projecte permet als estudiants dissenyar, programar i optimitzar un prototip funcional d'un videojoc de carreres tridimensional utilitzant un motor utilitzat de la indústria com **Unity**.
